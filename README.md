## ğŸš€ Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation
1. Clone the repository:
```bash
git clone git@github.com:FenWangSiCheng/Quiz.git
cd Quiz
```

2. Install dependencies for the quiz app:
```bash
cd sap-quiz-app
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```
